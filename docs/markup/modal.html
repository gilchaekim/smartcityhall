UI - Events

Attributes
Name Description
beforeshow 항목 실행되기 직전 발생합니다.
show 항목이 실행된 직후 발생합니다.
shown Show animation 이 완료된 후 발생합니다.
beforehide 항목이 닫히기 직전 발생합니다.
hide 항목이 닫힌 직후 직전 발생합니다.
hidden Hide animation 이 완료된 후 발생합니다.

UI - Modal Dialog Box

다이얼로그 공통

  then() 함수를 통해 콜백을 제공하고 있습니다.
Example -
 GCui.modal.alert({
   title:"String",
   text:"String",
   ok: "String"        // 확인 버튼 문구 (Defaut "확인")
 }).then(function(){console.log('Closed!')})
 
 
 GCui.modal.confirm({
   title:"String",
   text:"String",
   ok: "String"        // 확인 버튼 문구 (Defaut "확인")
   cancel: "String"    // 취소 버튼 문구 (Defaut "취소")
 }).then(function(){console.log('OK')}, function(){console.log('CANCEL')})
