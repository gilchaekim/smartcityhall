Attribute - Button

Attributes
Property Description Type Default
label 버튼 텍스트 string -
size 버튼 사이즈 : large, medium, small string medium
size 버튼 타입 : primary , success, grey, white, info, default string primary
isIcon 아이콘 포함 여부 Boolean false

Attribute - Button Group

Attributes
Property Description Type Default
btns 버튼 그룹 텍스트 리스트 array -
className 커스텀 클래스 string -
